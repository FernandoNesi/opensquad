# ANALYSER 14 LOW TICKET 3F3N

## O que é

Uma skill de auditoria de conversão focada em páginas low ticket.

Ela identifica:
- onde a página perde dinheiro
- quanto está perdendo
- o que corrigir primeiro

---

## Quando usar

- Página pronta ou quase pronta
- Antes de rodar tráfego
- Quando conversão está baixa
- Para priorizar otimizações

---

## Resultado esperado

Você sai com:

- diagnóstico claro
- ranking de problemas
- plano de ação direto

---

## Diferença para outras skills

Essa skill NÃO cria.

Ela decide.

---

## Fluxo ideal 3F3N

1. Criar página (Skill Estrutura)
2. Ajustar copy (Skill Copy)
3. Rodar essa skill (Analyzer)
4. Iterar com base no impacto

---

## Filosofia

Pare de otimizar no feeling.

Otimize onde tem dinheiro.


## **Lógica de blocos:** 14 blocos

**1. Bloco = Vender sozinho**

Objetivo = Vender o produto sozinho, pessoa bater o olho e já querer comprar; passar a

ideia central do produto; despertar a curiosidade. É para ser simples e direto ao ponto.

**2. Bloco = Depoimentos**

Provar o que foi dito no bloco 1; mostrar provas reais para persona; fazer a persona se

identificar com o antes. Quando não tem prova social, você remove, ou coloca alguma

prova teórica, uma comprovação de resultado.

**3. Bloco = Dor latente principal**

Objetivo principal: gerar identificação. Aumentar a dor/desejo da persona; lembrar ela do

problema; colocar o dedo na ferida para o próximo bloco

**4. Bloco = Transição da dor para solução**

Abrir a porta para oferecer o produto. Sair da dor/desejo para solução; mostrar para ela que

tem a solução; começar a venda do produto. É justamente por isso que você precisa do

[produto].

**5. Bloco = Passo a Passo da solução da dor**

Mostrar que é fast food. Mostrar que é rápido de resolver; mostrar que é simples, só seguir;

fazer a persona sentir que consegue fazer.

**6. Bloco = Tudo que você vai receber + Bônus**

Mostrar que ela vai chegar na promessa. Mostrar que com tudo isso ela chega no resultado;

fazer ela imagina que tudo isso vai ser caro; colocar bônus que ajuda no resultado. Ele

precisa ficar muito claro, para ela sentir que está recebendo várias coisas.

**7. Bloco = Para quem serve**

Fazer a persona sentir que é para ela. Fazer um checklist com várias dores, para pessoa se

identificar.

**8. Bloco = Ancoragem com preço**

Mostrar o quanto tudo isso vale. Assustar ela com o preço; mostrar que tem um desconto.

Tudo isso deveria custar X reais, porém.(bloco 9)

**9. Bloco = Valor (botão para comprar)**

Valor com desconto, com imagem com tudo que ela vai ganhar

**10. Bloco = Conversa séria**

Colocar a pessoa contra a parede. Lembrar ela das dores; fazer ela tomar decisão.

**11. Bloco = Autoridade**Fazer ela acreditar que ela achou a pessoa certa para ajudar. Focar na autoridade de

acordo com a promessa; fazer ela sentir confiança.

**12. Bloco = Valor (botão para comprar)**

Botão para comprar novamente. Repetir.

**13. Bloco = FAQ**

Perguntas frequentes. Objeções que a pessoa pode ter.

**14. Bloco = Rodapé**

Colocar logo, mostrar autoridade; deixar e-mail profissional; colocar CNPJ reforça a

confiança; lembre: é público frio
