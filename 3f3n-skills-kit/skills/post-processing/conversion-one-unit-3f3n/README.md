  
# CONVERSION ONE UNIT 3F3N

## 🎯 Objetivo

Resolver problemas de conversão focando em:

- decisão de compra
- redução de fricção
- aumento de clareza
- impacto financeiro direto

---

## ⚙️ Como funciona

1. Usuário fornece ideia ou elemento (página, copy, funil)
2. A skill identifica pontos de perda
3. Aplica análise baseada em:

- clareza → interesse → confiança → ação
- psicologia de decisão
- comportamento do usuário

4. Retorna diagnóstico + correção + execução

---

## 🧠 Por que funciona

Conversão não depende de design.

Depende de:

- clareza
- percepção de valor
- redução de risco

---

## 🔥 Quando usar

- página não converte
- baixa taxa de clique
- abandono no funil
- copy fraca

---

## 🚫 Quando não usar

- tarefas puramente técnicas
- design visual sem foco em venda

---

## 💣 Resultado

Identifica exatamente onde está perdendo dinheiro  
e corrige com execução direta
