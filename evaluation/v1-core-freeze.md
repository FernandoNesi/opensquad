# V1 Core Freeze

## Regra Absoluta

Congelar o core por 30 dias durante a finalização operacional da V1.

## Não Criar

- novas layers;
- novas filosofias;
- novos runtimes;
- novas governanças;
- novas abstrações;
- sistemas adaptativos;
- auto-evolução;
- novas arquiteturas.

## Só Permitir

- correção de comportamento observado;
- redução de exagero;
- remoção de burocracia;
- melhoria prática baseada em execução real.

## Critério De Mudança

Toda mudança futura precisa apontar para uma execução real em `evaluation/executions/`.

Sem execução real, sem mudança estrutural.

## Objetivo Da V1

Transformar o OpenSquad de framework inteligente em ferramenta operacional usada diariamente.
