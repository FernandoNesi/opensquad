# Evaluation & Calibration Report

## Regra Da Fase

O sistema deve parecer um operador experiente ajudando outra operação.

Não deve parecer uma empresa tentando provar inteligência.

## Objetivo

Medir comportamento real dos squads em cenários concretos antes de qualquer nova evolução cognitiva.

Durante esta fase, não criar:

- novo runtime;
- nova arquitetura;
- memória complexa;
- loops;
- sistemas distribuídos;
- novas layers cognitivas.
