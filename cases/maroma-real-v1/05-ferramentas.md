# Ferramentas

## Ferramentas Identificadas

- WhatsApp
- Sistema da franquia
- Planilhas
- Banco
- Instagram
- Controle manual
- Conversas verbais
- Possíveis grupos familiares
- Desenvolvimento App Multi Finança para lançar e gerenciar financeiro

## Problema Principal

As ferramentas provavelmente não funcionam como sistema integrado.

Elas funcionam como:
- pedaços soltos;
- memória distribuída;
- operação informal.
